## 🎓 使用场景

### 场景1: 基础@转发

```
用户: @BotA 请帮我分析数据
BotA: 收到请求，@BotB 来处理数据分析
BotB: [自动响应] 好的，开始分析...
```

### 场景2: 多机器人协作

```
用户: @BotA 帮我完成整个流程
BotA: @BotB 处理第一步，@BotC 处理第二步
BotB: [自动响应] 第一步完成
BotC: [自动响应] 第二步完成
```

### 场景3: 权限控制

```
BotA: @BotC 请求处理  ✅ 允许（在白名单）
BotB: @BotC 请求处理  ❌ 拒绝（不在白名单）
```
