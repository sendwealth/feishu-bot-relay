## 🎯 功能特性

### ✅ 核心功能

1. **消息监听与解析**
   - 支持飞书富文本格式
   - 智能提取@提及
   - 过滤用户@（只处理机器人）

2. **消息中转**
   - 一对一@提及
   - 一对多@提及
   - 保留完整上下文

3. **虚拟事件构造**
   - 符合飞书消息格式
   - 包含中转链信息
   - 支持嵌套中转

4. **机器人管理**
   - 注册/注销机器人
   - 状态管理
   - 权限控制

5. **安全机制**
   - 消息签名验证
   - 权限检查
   - 循环检测
   - 防重放攻击

### 🌟 高级特性

- **权限控制**: 细粒度的canRelayTo / canBeRelayedBy
- **循环检测**: 基于relay_chain + relay_count
- **RESTful API**: 完整的管理和测试接口
- **健康检查**: /health监控接口
- **日志记录**: 详细的操作日志
